# CodeRikkyo Doc

このドキュメントは、立教大学プログラミングサークル CodeRikkyoの教材です。
プログラミングを初めて勉強する学生向けに作成しております。

教材の制作はサークル員である学生が行なっております。
内容については十分精査しておりますが、誤りがある場合もございます。ご了承ください。

## カリキュラム

以下の言語について学習します。

- html/css/jquery
- Ruby/Ruby on Rails

月  | タイトル
--|--
4月| html/cssの基礎とサンプルページ制作
5月| Jqueryとアイディア出しと自主制作
